  <name>Task 2: Apply glass effects to container and cards</name>
  <files>views/css/style.css</files>
  <action>
Update the following selectors to use glass effects:

1. **.container** (~line 83): Change from solid to glass background
   - Replace `background: var(--bg);` with `background: var(--glass-bg-heavy);`
   - Add `backdrop-filter: blur(var(--glass-blur));`
   - Add `-webkit-backdrop-filter: blur(var(--glass-blur));` (Safari/older Webkit)
   - Update box-shadow to use glass-shadow token

2. **.card** (~line 311): Apply glass to session cards
   - Replace `background: var(--card-bg);` with `background: var(--glass-bg);`

   - Add `-webkit-backdrop-filter: blur(var(--glass-blur-light));`
   - Replace `border: 1px solid var(--card-border);` with `border: 1px solid var(--glass-border);`

3. **.card:hover** (~line 326): Update hover state
   - Replace `background: var(--card-hover);` with `background: var(--glass-bg-light);`
   - Update border to `border-color: var(--glass-border);`

4. **.pinned-item** (~line 208): Apply glass to pinned path items
   - Replace `background: var(--card-bg);` with `background: var(--glass-bg);`
   - Add `backdrop-filter: blur(var(--glass-blur-light));`
   - Add `-webkit-backdrop-filter: blur(var(--glass-blur-light));`
   - Replace border with `border: 1px solid var(--glass-border-light);`

5. **.pinned-item:hover** (~line 224): Update hover
   - Replace `background: var(--card-hover);` with `background: var(--glass-bg-light);`

IMPORTANT: Always include both `backdrop-filter` AND `-webkit-backdrop-filter` for compatibility.
